# Professional Projects

## Project 1: Golden AMI Pipeline
**Company/Role**: Maximus Inc - Platform Engineer  
**Duration**: 6 months  
**Technologies**: ["jenkins", "ec2-image-builder", "groovy", "aws", "automation", "security"]  
**Tags**: ["security", "automation", "compliance", "enterprise"]

### Context
Need to automate the creation and maintenance of hardened, compliant Amazon Machine Images (AMIs) for enterprise workloads while ensuring security compliance and removing manual processes.

### Solution
Engineered a comprehensive Golden AMI pipeline leveraging Jenkins orchestration, EC2 Image Builder for automated image creation, and Groovy shared libraries for reusable pipeline components. Implemented automated monthly hardened image creation with built-in compliance validation workflows.

### Impact
- Eliminated 100% of manual effort in AMI creation and maintenance processes
- Improved security posture through consistent, automated hardening procedures
- Ensured continuous compliance validation without manual intervention
- Reduced time-to-deployment for new instances through pre-hardened images
